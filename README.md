# Mini RDBMS (Pesapal Junior Dev Challenge 2026)

This project is a simple relational database management system (RDBMS)
implemented from scratch as part of the Pesapal Junior Dev Challenge 2026.

The goal is to demonstrate understanding of core database concepts such as:
- Tables and schemas
- CRUD operations
- Primary and unique keys
- Indexing
- Basic SQL parsing
- Joins
- Persistence
